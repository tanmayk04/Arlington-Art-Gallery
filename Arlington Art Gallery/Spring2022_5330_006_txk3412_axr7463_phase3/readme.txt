Please follow below steps to run web interface to perform diffent operations on database.

1) Copy Demo folder from .zip file to xampp/htdocs folder
2) Start the xampp server.
3) For View: Open any browser and browse http://localhost/Demo/View.html
4) For Insert: Open any browser and browse http://localhost/Demo/Insert.html
5) For Update: Open any browser and browse http://localhost/Demo/Update.html
6) For Delete: Open any browser and browse http://localhost/Demo/Delete.html
